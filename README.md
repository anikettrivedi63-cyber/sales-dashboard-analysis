# Sales Dashboard Analysis 📊

## 📌 Project Overview
This project analyzes sales data to identify key business insights such as revenue trends, top-performing products, and regional performance.

## 🎯 Objective
- Analyze sales performance
- Identify profitable categories
- Understand regional trends

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

## 📊 Key Insights
- West region generated highest sales
- Technology category is most profitable
- Sales increase during year-end months

## 📁 Dataset
- Contains 200+ sales records
- Includes region, category, product, sales, and profit
